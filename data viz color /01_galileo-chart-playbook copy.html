Better — clearer structure

Simplify before adding more encoding.

08 Glossary
MetricThe value being measured.
DimensionThe field used to group a metric.
Dimension memberOne value inside a dimension.
SeriesA related set of values shown in a chart.
MarkThe visual form representing data.
Value natureWhether a value is observed or estimated.
Time windowThe period used to calculate a value.
ScenarioA possible outcome under assumptions.
MethodHow a value was calculated or assigned.
ProvenanceWhere a value or recommendation came from.
