05 Current Galileo modifier defaults
Modifier Current default Guidance
Actual
Primary · solid
 Observed value. Use Primary with full solid stroke or fill.
Forecast
Primary + Primary-A
 Primary-A may reinforce Forecast, but must not be the only distinction on thin marks. For thin lines, add dash, markers, or stroke weight.
If Primary-A lacks contrast, retain Primary and use dash.
Trailing
Metric family + reduced treatment
 Trailing changes the time window, not the categorical identity. Retain the relevant Primary color family. Distinguish through line treatment and reduced emphasis.
Do not assign an unrelated categorical color to Trailing series.
Attributed
Primary + Primary-B
 Primary-B may reinforce Attributed, but should not carry the meaning alone when ambiguity exists. Add a label or supporting mark treatment.
Confirm whether Attributed is the metric or the method before assigning color.
Model recommendation
Metric identity + AI emphasis
 Preserve the underlying metric or categorical identity. Add AI accent when recommendation provenance matters. Do not replace the metric identity with AI purple.
AI accent communicates provenance or recommendation emphasis.
Pacing data Dedicated pacing semantic colors Pacing remains separate from categorical color identity and tonal variants. Use the dedicated pacing semantic color system.

These are current Galileo defaults. Final application depends on chart mark, chart density, and legibility.

06 Attributed Sales vs Attributed modifier
Confirm whether Attributed is the metric or the method before assigning color.
Attributed Sales as metric
Primary identity: Attributed Sales
Color: charts-cat-3 · #7DC50F (approved metric color)
Attributed Sales has its own approved Primary color. Treat it as a core metric identity.
Attributed as method modifier
Primary identity: Spend (metric)
Data modifier: Attributed (method)
Treatment: Primary-B + square marker + label
Attributed as a method retains the underlying metric identity. Use Primary-B plus an appropriate mark treatment.

11 Compact reference
Primary identityThe main metric, dimension member, or categorical identity represented by the Primary color family.
Value natureWhether a value is observed or estimated.
Time windowThe period or rolling range used to calculate a value.
MethodHow a value was calculated or assigned.
ScenarioA possible outcome under assumptions.
ProvenanceWhere a value or recommendation came from.
Semantic statusA dedicated evaluative state such as pacing.
